Skills & Tech Stack

HTML

HTML is the foundational language for creating web pages. It provides the structure and layout for content on the internet.

CSS

CSS is responsible for the visual presentation of a web page. It controls the layout, color, typography, and overall design.

Javascript

JavaScript is a versatile programming language that enables dynamic and interactive elements on websites.

Bootstrap

Bootstrap is a popular front-end framework that provides a collection of pre-built design components and responsive layouts.

Node JS

Node.js is a powerful runtime environment for executing JavaScript on the server side.

Git

Git is a distributed version control system that enables developers to track and manage changes in their codebase efficiently.

APIs

API integration involves connecting different software systems to enable communication and data exchange.

React

React is a widely used JavaScript library for building user interfaces, particularly single-page applications.
